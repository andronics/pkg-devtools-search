# Search Tools - File and content search utilities

**Package**: `@devtools/search`
**Version**: 0.1.0
**Repository**: `pkg-devtools-search`

## Overview

Search Tools - File and content search utilities

## Installation

### Prerequisites

This package requires the following external commands:

- `dmenu`

### Using pkg-cli

```bash
# Install from GitHub
pkg-cli install @devtools/search

# Or install from local source
cd ~/.pkgs
stow search
```

## Usage

### Development Tool

Search Tools - File and content search utilities


## Configuration

Configuration files are typically stored in:
- `~/.config/search/` - User configuration
- `~/.local/share/search/` - Application data

## Uninstallation

```bash
# Using pkg-cli
pkg-cli uninstall @devtools/search

# Or manual unstow
cd ~/.pkgs
stow -D search
```
